# AI Image Classification Web App

This repository contains an AI-powered image classification web application built using Streamlit and a fine-tuned MobileNetV2 model.

## Files
* `app.py`: The main Streamlit application code.
* `model.keras`: The trained MobileNetV2 model.
* `class_names.json`: Mapping of model output indices to string labels.
* `requirements.txt`: Dependencies required to run the app.

## Frameworks Used
* TensorFlow / Keras
* Streamlit
